## Getting started

Clone o projeto com:

<pre>git clone https://github.com/rafaelafpro/Desafio-Fetch-Countries.git</pre>

Instale o a biblioteca live-server com:

<pre>npm install live-server</pre>

Vá até o diretório e execute o comando:

<pre>live-server</pre>
